O problema foi desenvolvido em python utilizando do VSC, caso seja executado na mesma plataforma o método de execução é Crtl+F5
caso contrário utilizar o command line "python3 main.py"
O programa irá pedir a entrada de dados das peças disponíveis e a posição bonus, após isso irá executar e se iniciará uma nova rodada
de jogadas com uma nova chamada de peças e posição, o programa irá encer caso a posição bonus não for um número inteiro ou se
houver corte de teclado "crtl+c" por exemplo
O programa executa a chamada do aqui Banco de palavras.txt, que deve estar na mesma pasta que os demais arquivos, de preferencia e
que apresenta apenas uma palavra por linha


exemplos de entrada:

0

0

2
